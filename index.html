💡 Comment fonctionne l'envoi

Une fois le bon signé, appuie sur « Envoyer par mail + Archiver ». L'email du client est automatiquement copié dans le presse-papier, et l'iPhone ouvre le menu de partage avec le PDF signé déjà prêt.

Choisis Mail dans ce menu : l'app Mail s'ouvre avec le PDF en pièce jointe. Fais un appui long sur le champ « À » puis Coller pour récupérer l'adresse du client automatiquement. Puis appuie sur Envoyer.

Le bon est archivé automatiquement dans l'onglet Historique dès que tu appuies sur le bouton.
